# Changelog

## 1.0.1-beta - 2026-06-22

### Fixed

- Long titles and summaries now wrap automatically by default instead of overflowing the panel.

### Added

- New setting for long text behaviour: automatic wrapping or horizontal scrolling.

## 1.0.0-beta - 2026-06-22

### Added

- Initial public release as `local_floatingnews`.
- Floating news panel for Moodle.
- News item management page.
- Image, title, short text, link and custom button support.
- Fade carousel, horizontal carousel, vertical rolling list and scrollable feed layouts.
- Custom colour and layout settings.
- Active/inactive status toggle from the management table.
- 1-based ordering with automatic reordering.
- Maximum active item limit with automatic deactivation beyond the configured limit.
- Front page only or site-wide display option.
- Guest visibility setting.
- Collapsed mode and remember-close option.
- Admin/manager shortcut label.
